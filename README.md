# Banks — Web Banking Application

A learning project: a full-stack web banking platform with user registration, OTP verification, secure login, money transfers, and a digital customer service interface.

## Stack

- **Backend:** Node.js, Express, TypeScript, MongoDB (Mongoose), JWT auth
- **Frontend:** React, TypeScript, Material UI
- **Real-time:** Socket.IO
- **Containerization:** Docker, docker-compose
- **CI/CD:** Jenkins
- **Testing:** Jest, Supertest, Cypress

## Project Status

In active development — built phase by phase as a learning project.

## Structure

- `docs/` — API specification, sequence diagrams
- `backend/` — Express + TypeScript server
- `frontend/` — React + TypeScript app
